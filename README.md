# AI-Based Network Traffic Analyzer

## Описание

Анализатор сетевого трафика на основе ИИ — это решение в области кибербезопасности, использующее машинное обучение для анализа сетевого трафика и обнаружения угроз, таких как сканирование портов, атаки типа «отказ в обслуживании» и утечка данных. Он автоматизирует проверку трафика, составление отчетов и оповещения, помогая организациям заблаговременно защищать свои сети.

---

## Возможности

- **Захват трафика**: Захват пакетов данных с помощью scapy для дальнейшего анализа.
- **Разработка признаков**: Извлечение и преобразование данных пакетов (IP-адрес, порт, протокол, длина) для использования в машинном обучении.
- **Обнаружение с помощью машинного обучения**: Обучает классификатор Random Forest для различения безопасного и вредоносного трафика с использованием размеченных наборов данных.
- **Анализ в реальном времени и пакетный анализ**: Поддерживает как мониторинг пакетов в реальном времени, так и анализ сохраненных файлов `.pcap` или `.csv`.
- **Оповещение и отчетность**: Выявляет подозрительные пакеты/сессии и формирует отчет на основе полученных данных для принятия мер.

---

## Как это работает

1. **Сбор данных**: Используйте Scapy для перехвата пакетов или импорта журналов трафика.
2. **Предварительная обработка**: Анализ необработанного трафика для извлечения характеристик (IP-адрес, порт, протокол, длина пакета).
3. **Обучение модели**: Обучите модель случайного леса, используя размеченные данные.
4. **Обнаружение**: Анализ нового трафика; модель выявляет подозрительную активность.
5. **Отчетность**: Создайте файл оповещений, содержащий список всего отмеченного трафика.

## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/Gladfn/ai-traffic-analyzer.git
   cd ai-network-analyzer
   ```
2. Запуск виртуального окружения
   2.1 Linux
      ```bash
      python -m venv venv
      source venv/bin/activate
      ```
   2.2 Windows
      ```bash
      python -m venv venv
      venv\Scripts\activate.bat
      ```

3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

**Нормализация трафика**
```bash
python src/preprocess.py --input "Трафик который надо нормализовать" --output data/features.csv
```

**Пример анализа тарфика:**
```bash
python src/preprocess.py --input "Пример вредоносного трафика" --output data/features.csv
python src/train_model.py --input data/features.csv --model data/model.pkl
python src/detect.py --model data/model.pkl --input "Трафик для анализа после нормализации" --output data/alerts.csv
```

**Запуск захвата трафика:**
```bash
sudo python src/capture.py
```
